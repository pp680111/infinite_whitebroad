## 卡片编辑功能测试用例

### 前提条件
- 应用正常运行，画布可以正常工作

---

### Test Case 1: 编辑内容后点击画布空白区域
**操作步骤：**
1. 双击画布空白区域，创建卡片A
2. 双击画布另一个位置，创建卡片B
3. 双击卡片A进入编辑状态
4. 在卡片A的内容区域输入文字（如 "Test Content"）
5. 点击画布空白区域（不是其他卡片）

**预期结果：**
- 卡片A退出编辑状态
- 输入的内容被保存
- 卡片A显示刚才输入的内容

---

### Test Case 2: 拖动卡片后创建新卡片
**操作步骤：**
1. 双击画布空白区域，创建卡片A
2. 双击画布另一个位置，创建卡片B
3. 双击卡片A进入编辑状态
4. 在卡片A中输入文字（如 "Card A Content"）
5. 点击画布空白区域退出编辑
6. 拖动卡片B到新位置（如向右拖动200px）
7. 双击画布空白区域创建新卡片C

**预期结果：**
- 卡片B保持在拖动后的新位置（不是原始位置）
- 卡片A的内容被保留

---

### Test Case 3: 点击其他卡片切换编辑
**操作步骤：**
1. 双击画布创建卡片A
2. 双击画布创建卡片B
3. 双击卡片A进入编辑状态
4. 在卡片A中输入文字
5. 点击卡片B

**预期结果：**
- 卡片A退出编辑状态，内容被保存
- 卡片B进入编辑状态
- 卡片A不显示编辑光标

---

### Test Case 4: 编辑后可以拖动其他卡片
**操作步骤：**
1. 双击画布创建卡片A
2. 双击画布创建卡片B
3. 双击卡片A进入编辑状态
4. 在卡片A中输入文字
5. 点击画布空白区域退出编辑
6. 尝试拖动卡片B

**预期结果：**
- 卡片A内容被保存
- 卡片B可以被正常拖动

---

### Test Case 5: 拖动卡片后位置同步
**操作步骤：**
1. 双击画布创建卡片A
2. 拖动卡片A到新位置
3. 双击画布创建卡片B（触发重新渲染）
4. 检查卡片A的位置

**预期结果：**
- 卡片A保持在拖动后的新位置

---

### Test Case 6: 编辑内容后直接拖动其他卡片
**操作步骤：**
1. 双击画布创建卡片A
2. 双击画布创建卡片B
3. 双击卡片A进入编辑状态
4. 输入 "Test" 文字
5. 不点击画布空白区域，直接拖动卡片B到新位置

**预期结果：**
- 卡片A显示 "Test" 文字
- 卡片B可以被正常拖动
